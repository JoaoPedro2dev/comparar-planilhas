# Comparador de Planilhas JS

Sistema web em JavaScript para comparar duas planilhas Excel, cruzar colunas, agrupar resultados e exportar para Excel.

## Funcionalidades
- Upload de 2 arquivos Excel
- Comparação entre colunas
- Tabela de resultado
- Total geral
- Exportação de resultados para Excel

## Tecnologias
- HTML
- CSS
- JavaScript
- SheetJS (XLSX)
